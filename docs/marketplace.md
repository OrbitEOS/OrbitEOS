# Marketplace

The OrbitEOS Marketplace extends the platform with plugins across nine categories.

## Categories

- **Energy Devices** — Device drivers for inverters, meters, batteries, EV chargers
- **Cloud Integrations** — Connect to weather APIs, energy markets, cloud services
- **Analytics** — Advanced energy analytics and reporting modules
- **Control Strategies** — Peak shaving, self-consumption, demand response algorithms
- **Notifications** — Email, SMS, webhook, and push notification channels
- **Dashboard Widgets** — Custom visualizations for the energy dashboard
- **Data Export** — CSV, API, and database export connectors
- **Protocols** — Modbus TCP, SunSpec, OCPP, and more
- **UI Apps** — Custom user interface applications

## How It Works

1. **Browse** — Explore available plugins by category, type, or search
2. **Install** — Install a plugin for your account with one click
3. **Assign** — Assign the plugin to specific sites with site-level configuration
4. **Configure** — Override settings per site (e.g., device IP address, API keys)

## Plugin Types

- **Device Drivers** — Connect hardware: SMA, Fronius, Huawei, KEBA, Janitza, and more
- **Cloud Integrations** — Weather data, electricity pricing, carbon intensity
- **Analytics** — Energy performance, degradation analysis, forecasting
- **Controllers** — Automated optimization strategies
- **Notifications** — Alert routing to any channel
- **Widgets** — Custom dashboard components

## Tools

- **`browse_plugins`** — Search and filter available plugins
- **`get_plugin_details`** — View configuration schema and capabilities
- **`install_plugin`** — Install a plugin for your account
- **`assign_plugin_to_site`** — Deploy a plugin to a specific site
- **`list_site_plugins`** — See what's installed on a site
