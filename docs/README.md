# OrbitEOS MCP Server

Energy management platform — 61 tools for solar, battery, grid, and EV charging control

`v1.0.0` `61 Tools` `OAuth 2.1` `Streamable HTTP`

## What is OrbitEOS?

OrbitEOS is an energy management platform that monitors and controls solar panels, batteries, grid connections, and EV chargers in real time. The MCP server exposes 61 tools across 9 categories, enabling AI assistants to query live energy data, analyze historical trends, manage devices, and control energy systems.

## Features

- **Live Monitoring** — Real-time power flow: grid, solar production, battery SoC, consumption
- **Historical Analytics** — Time-series power data and cumulative energy totals over any date range
- **Diagnostics** — System health checks, grid usage analysis, solar comparison, battery sizing
- **Device Control** — Create, update, delete components; write to channels (EV charger limits, battery modes)
- **Multi-Tenant** — Accounts, sub-accounts, site hierarchy (region/hub/site/zone), RBAC
- **User Management** — Create users, assign roles, manage permissions, invite via email
- **Marketplace** — Browse/install plugins for device drivers, analytics, control strategies
- **Audit & Logs** — System logs, audit trail, platform health monitoring
- **CSV Export** — Export historical data for external analysis
