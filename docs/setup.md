# Setup

## Claude.ai / Claude Desktop

1. Go to **Settings > Connectors**
2. Click **Add custom connector**
3. Enter URL: `https://mcp.orbiteos.cloud/mcp`
4. Click **Add** — OAuth will prompt you to log in

## Claude Code (CLI)

```bash
claude mcp add --transport http orbiteos https://mcp.orbiteos.cloud/mcp
```
