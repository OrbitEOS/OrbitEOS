# Marketplace

De OrbitEOS Marketplace breidt het platform uit met plugins in negen categorieën.

## Categorieën

- **Energieapparaten** — Apparaatdrivers voor omvormers, meters, batterijen, laadpalen
- **Cloud-integraties** — Verbinding met weer-API's, energiemarkten, clouddiensten
- **Analytics** — Geavanceerde energieanalyse- en rapportagemodules
- **Besturingsstrategieën** — Piekvermijding, eigenverbruik, vraagsturingsalgoritmen
- **Meldingen** — E-mail, SMS, webhook en push-notificatiekanalen
- **Dashboard Widgets** — Aangepaste visualisaties voor het energiedashboard
- **Data Export** — CSV, API en database-exportconnectoren
- **Protocollen** — Modbus TCP, SunSpec, OCPP en meer
- **UI Apps** — Aangepaste gebruikersinterface-applicaties

## Hoe het werkt

1. **Bladeren** — Verken beschikbare plugins op categorie, type of zoekterm
2. **Installeren** — Installeer een plugin voor uw account met één klik
3. **Toewijzen** — Wijs de plugin toe aan specifieke sites met sitespecifieke configuratie
4. **Configureren** — Overschrijf instellingen per site (bijv. apparaat-IP-adres, API-sleutels)

## Plugin types

- **Apparaatdrivers** — Sluit hardware aan: SMA, Fronius, Huawei, KEBA, Janitza en meer
- **Cloud-integraties** — Weerdata, elektriciteitsprijzen, koolstofintensiteit
- **Analytics** — Energieprestaties, degradatieanalyse, voorspellingen
- **Controllers** — Geautomatiseerde optimalisatiestrategieën
- **Meldingen** — Waarschuwingsroutering naar elk kanaal
- **Widgets** — Aangepaste dashboardcomponenten

## Tools

- **`browse_plugins`** — Beschikbare plugins zoeken en filteren
- **`get_plugin_details`** — Configuratieschema en mogelijkheden bekijken
- **`install_plugin`** — Een plugin voor uw account installeren
- **`assign_plugin_to_site`** — Een plugin op een specifieke site inzetten
- **`list_site_plugins`** — Bekijken wat op een site geïnstalleerd is
