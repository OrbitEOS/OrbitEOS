# Installatie

## Claude.ai / Claude Desktop

1. Ga naar **Instellingen > Connectors**
2. Klik op **Aangepaste connector toevoegen**
3. Voer URL in: `https://mcp.orbiteos.cloud/mcp`
4. Klik op **Toevoegen** — OAuth vraagt u om in te loggen

## Claude Code (CLI)

```bash
claude mcp add --transport http orbiteos https://mcp.orbiteos.cloud/mcp
```
