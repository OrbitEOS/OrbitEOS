# OrbitEOS MCP Server

Energiemanagementplatform — 61 tools voor zon, batterij, net en laadpaalbesturing

`v1.0.0` `61 Tools` `OAuth 2.1` `Streamable HTTP`

## Wat is OrbitEOS?

OrbitEOS is een energiemanagementplatform dat zonnepanelen, batterijen, netaansluitingen en laadpalen in realtime bewaakt en bestuurt. De MCP-server biedt 61 tools verdeeld over 9 categorieën, waarmee AI-assistenten live energiedata kunnen opvragen, historische trends analyseren, apparaten beheren en energiesystemen aansturen.

## Functies

- **Live Monitoring** — Realtime energiestromen: net, zonneproductie, batterij SoC, verbruik
- **Historische Analyse** — Tijdreeksdata en cumulatieve energietotalen over elke periode
- **Diagnostiek** — Systeemgezondheidscontroles, netverbruikanalyse, zonvergelijking, batterijdimensionering
- **Apparaatbesturing** — Componenten aanmaken, bijwerken, verwijderen; kanaalwaarden schrijven (laadpaalbegrenzingen, batterijmodi)
- **Multi-Tenant** — Accounts, sub-accounts, sitehiërarchie (regio/hub/site/zone), RBAC
- **Gebruikersbeheer** — Gebruikers aanmaken, rollen toewijzen, rechten beheren, uitnodigingen per e-mail
- **Marketplace** — Plugins installeren voor apparaatdrivers, analytics, besturingsstrategieën
- **Audit & Logs** — Systeemlogboeken, audittrail, platformgezondheidsmonitoring
- **CSV Export** — Historische data exporteren voor externe analyse
